# GPT-4 Technical Report

**Authors:** OpenAI (Josh Achiam, Steven Adler, Sandhini Agarwal, Lama Ahmad, Ilge Akkaya, Florencia Leoni Aleman, Diogo Almeida, Janko Altenschmidt, Sam Altman, Shyamal Anadkat, et al.)
**Affiliation:** OpenAI

## Publication Status

- **arXiv preprint:** March 2023, arXiv:2303.08774
- **Peer-reviewed:** No
- **Conference/Journal:** None
- **Status:** Preprint

The GPT-4 Technical Report was released alongside the model's API launch. It remains unpublished at a peer-reviewed venue. Notably, the report discloses almost no details about model architecture, training data, or training compute, citing competitive and safety concerns.
