# SmolLM3: smol, multilingual, long-context reasoner

**Authors:** Elie Bakouch, Carlos Miguel Patino, Anton Lozhkov, Edward Beeching, Aymeric Roucher, Nouamane Tazi, Aksel Joonas Reedi, Guilherme Penedo, Hynek Kydlicek, Clementine Fourrier, Nathan Habib, Kashif Rasul, Quentin Gallouedec, Hugo Larcher, Mathieu Morlon, Joshua Lochner, Vaibhav Srivastav, Xuan-Son Nguyen, Colin Raffel, Lewis Tunstall, Loubna Ben Allal, Leandro von Werra, Thomas Wolf
**Affiliation:** Hugging Face

## Publication Status

- **Published:** July 8, 2025, Hugging Face blog post
- **Peer-reviewed:** No
- **Conference/Journal:** None. This is a technical blog post accompanying the model release, not a formal paper. There is no arXiv preprint.
- **Status:** Informal community contribution

SmolLM3 is the third iteration of the SmolLM series. The predecessor SmolLM2 has a formal arXiv paper (Allal et al., arXiv:2502.02737), but SmolLM3 was released solely as a blog post and model weights on Hugging Face Hub.
