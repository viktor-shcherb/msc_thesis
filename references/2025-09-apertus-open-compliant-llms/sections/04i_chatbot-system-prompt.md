# 4.4 Chatbot System Prompt [p. 36]

[p. 36] At the final step, Apertus's system prompt provides more specific instructions and guidance depending on the specific task and setting. Appendix P includes a recommended system prompt for deploying Apertus as a general-use chatbot. The prompt gives summary information on the bot's identity, training origin, and capabilities. It also includes a summary version of the Swiss AI Charter (Appendix O) and some extra instructions on context and style.
